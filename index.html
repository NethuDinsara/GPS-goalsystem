Quick Connection Guide (Path A)

  1. Create a fresh Google Sheet. Set up 3 tabs: Goals & Plans, Daily_Systems, and Weekly_Reviews.
  2. Click Extensions > Apps Script in your Sheet menu.
  3. Delete any template code and paste your custom Google Script router code.
  4. Click the blue Deploy button (Top right) and select New Deployment.
  5. Select Web App. Set Execute as: Me and Who has access: Anyone.
  6. Deploy, authorize access, copy the generated Web App URL, and paste it into the input box above!
